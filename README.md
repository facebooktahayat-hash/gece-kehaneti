# Gece Kehaneti — Next.js + Tailwind

Lovable bağımsız ilk sürüm. Bu proje karanlık/neon temalı fal sitesi ön yüzünü içerir.

## İçerik
- Ana sayfa
- Kategoriler
- Paket/ürün detay sayfaları
- Sipariş formu
- Ödeme ekranı
- Kullanıcı paneli görünümü
- Admin paneli görünümü
- Hukuki sayfalar
- Mobil uyumlu karanlık/neon tasarım

## Kurulum
```bash
npm install
npm run dev
```

Sonra tarayıcıda: `http://localhost:3000`

## Yayınlama
1. GitHub'a yükle
2. Vercel hesabına gir
3. New Project → GitHub reposunu seç
4. Deploy
5. Domain bağla
