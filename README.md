# Gece Kehaneti

Next.js + Tailwind tabanlı Gece Kehaneti projesi.

## Güncel risk azaltma düzeni

- Ürün bedelleri TL yerine Gece Kredisi olarak gösterilir.
- Kredi oranı: 1 TL = 1 Gece Kredisi.
- Kehanet özel fiyatı: 25.000 Gece Kredisi.
- Diğer paketlerde taban 500 Gece Kredisi ve yaklaşık %25 kademeli artış kullanılır.
- Yorum talep formunda telefon zorunlu değildir.
- Doğum tarihi yalnızca astroloji ve numeroloji için zorunludur.
- Fotoğraf yalnızca kahve falı için zorunludur.
- Admin linki footer’dan kaldırılmıştır; /admin varsayılan olarak 404 döner.
- Gece Kredisi site içi kullanım kredisi olarak açıklanır; devredilmez ve nakde çevrilmez.

## Kurulum

```bash
npm install
npm run build
npm run dev
```
